（1）制作サイトのタイトル
KASA REMIND - 傘忘れ防止LINEアプリ

（2）制作サイトの説明
OpenWeatherMap APIとLINE Messaging APIを使った、雨の日だけLINEで通知する傘忘れ防止Webアプリです。  
「天気予報を確認し忘れて雨に降られた」「雨が止んだあと傘を置き忘れた」という日常のストレスを自動通知で解決します。  
晴れの日は一切通知しないため、通知が煩わしくなる問題も防いでいます。

通知は3つのタイミングで送られます。前日夜には翌日の雨を事前にお知らせし、当日朝にはお出かけ前の最終確認を送ります。
日中に雨が止んだ場合は、傘の置き忘れを防ぐためのリマインダーを夕方まで定期的に送ります。

（3）工夫した点・こだわった点
- 雨の日だけ通知する仕組みにこだわりました。晴れの日に通知が来ると邪魔になって結局無視してしまうため、降水確率が閾値を超えた日のみ送信するロジックを実装しています
- 通知時間やリマインダーの間隔をブラウザ上のUIで自由に変更・保存できるようにしました。人によって生活リズムが違うため、自分に合ったスケジュールにカスタマイズできます
- 薄い水色とホワイトを基調にした落ち着いたデザインにしました。毎日使うツールなので、圧迫感のない見やすい画面を意識しています
- `fetch()` を使って外部APIからデータを取得し、取得したオブジェクトのプロパティ（`item.pop` など）にアクセスして表示する、授業で学んだ流れをそのまま実装しています

（4） 難しかった点・次回トライしたいこと
- OpenWeatherMapの無料プランは3時間ごとのデータしか取得できないため、より細かい1時間ごとの予報を使うには有料プランが必要です。次回は有料プランまたは別の天気APIも検討したいです
- 現状はページを開いている間しか自動通知が動作しません。常時バックグラウンドで動かせる仕組みも次のステップとして挑戦したいです
